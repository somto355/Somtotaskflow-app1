# TaskFlow App

## 🚀 Live Demo
https://somtotaskflow-app1-5sby.vercel.app/


## 📌 Description
A task management app that allows users to:
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed

## 🛠️ Tech Stack
- React
- JavaScript
- CSS

## ✨ Features
- Clean UI
- Responsive design
- State management



## ⚙️ Installation
npm install
npm start
